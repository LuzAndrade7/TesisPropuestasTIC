**Historial de Cambios**

## [1.0.0] - 2024

### Añadido
- Estructura completa del proyecto con arquitectura en capas
- Backend .NET 8 con ASP.NET Core Web API
- Frontend Angular 17 con Bootstrap 5
- Modelo de base de datos PostgreSQL
- Controllers para CRUD de propuestas
- Services implementando lógica de negocio
- Repositories para acceso a datos
- DTOs para transferencia de datos
- Formularios reactivos en Angular
- Componentes de lista, detalle, crear y editar propuestas
- Servicio de consumo de API desde Angular
- Integración CORS
- Swagger para documentación de API
- Docker y docker-compose para despliegue
- Scripts SQL para inicialización de BD
- Documentación completa

### Características iniciales
- Gestión completa de propuestas de TIC
- Estados de propuesta (Pendiente, Enviada, En Revisión, Aprobada, Rechazada, Observada)
- Asignación de estudiantes a propuestas
- Validaciones en cliente y servidor
- Manejo de errores consistente
- Interfaz responsiva
