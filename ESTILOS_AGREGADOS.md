## 🔧 SI ALGO SE VE MAL

### "Los estilos no se cargan"
```
1. F12 → Presiona Ctrl+Shift+R
2. Abre Network tab
3. Busca estilos .css
4. Si están en rojo (404), recarga la app
```

### "Se ve diferente en móvil"
```
Los estilos móviles están configurados:
- @media (max-width: 768px) — Tablet
- @media (max-width: 480px) — Mobile
```

### "Quiero cambiar un color"
```
Variables en :root en styles.css:
--navy: #0E2240;
--gold: #F3BD46;
← Cambiar aquí y aplica globalmente
```
