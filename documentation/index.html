🚀 Quick Start

Backend (ASP.NET Core 8.0)

cd backend
dotnet build                    # Compilar
dotnet run                      # Ejecutar en http://localhost:5000

Frontend (Angular 17)

cd frontend
npm install                     # Primera vez
npm start                       # Ejecutar en http://localhost:4200

Accesos

🔗 API Swagger (localhost:5000/swagger) 🔗 Frontend (localhost:4200) 📡 Postman Collection

🔄 Flujo de Estados (Diagrama Simplificado)

BORRADOR
   ↓ [Enviar revisión]
PENDIENTE
   ├─ [Revisar OK] → APROBADA
   │              ├─ [Asignar estudiantes] → PENDIENTE (nueva revisión)
   │              └─ [Solicitar nueva aprobación] → PENDIENTE
   │
   └─ [Observación] → OBSERVADA
                      ├─ [Editar] → OBSERVADA
                      └─ [Reenviar] → PENDIENTE

BORRADOR → Eliminar ✅ (permitido)
Otros estados → Eliminar ❌ (prohibido)
